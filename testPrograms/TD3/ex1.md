2. En une phrase, que fait ce programme ?
Ce programme calcule la somme de tous les entiers naturels inférieurs ou égaux à n.

n = 0
somme = 0

n = 1
somme = 0 + 1 = 1

n = 2
somme = 1 + 2 = 3

n = 5
somme = 1 + 2 + 3 + 4 + 5 = 15
